<h1 align = "center"> ☕ Coffee Shop App </h1>

A fully functional mobile application for managing a coffee shop, built using **Kotlin** for Android.  
The app provides a seamless experience for browsing coffee, placing orders, tracking deliveries, and managing a shopping cart, all with a clean and responsive UI.

---

## 🌟 Features

- 🏠 **Home Page** – Browse featured coffee and categories  
- 🔍 **Explore Coffee** – View and search the full coffee menu  
- 📄 **Details Page** – Check detailed descriptions of coffee items  
- 🛒 **Cart** – Add, remove, and adjust orders  
- ✅ **Checkout Page** – Place orders and handle payment (mock or integrated)  
- 🔐 **Login / Authentication** – Secure user login and registration  
- 🚚 **Track Order Page** – Track the status of ongoing orders in real-time  

---
